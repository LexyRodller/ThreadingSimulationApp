# Класс «Стек»
### Свойства:  
- текущий элемент (current item);
- количество элементов (number of items);
- признак пуст ли список (the list is empty).
### Методы:  
- добавления элементов в вершину стека (push);
- удаление элемента с вершины стека (pop);
- очистка стека (clear).
